Installation
------------

- The backend layouts of Bootstrap Package (bootstrap_package) must be loaded
- The static template of Bootstrap Package (bootstrap_package) must be loaded before this one
- The static template of Bootstrap3 responsive images (lef_responsive_images) must be loaded before this one.

To install the extension, perform the following steps:

#. Go to the Extension Manager
#. Install the extension
#. Load the static template after Bootstrap Package (bootstrap_package) and after Bootstrap3 responsive images (lef_responsive_images)
#. ...

Configuration
-------------

There is nothing to do.
